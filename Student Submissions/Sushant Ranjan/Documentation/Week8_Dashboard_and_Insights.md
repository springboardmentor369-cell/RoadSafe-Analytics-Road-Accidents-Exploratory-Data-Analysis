# Week 8 - Dashboard Development and Insights

## Objective

The objective of this week was to build an interactive Power BI dashboard and derive meaningful insights from the data.

## Activities Performed

* Designed dashboard layout for better visualization
* Created charts and visuals (bar charts, line charts, KPI cards)
* Added filters and slicers for interactivity
* Integrated all KPIs into the dashboard

## Dashboard Features

* Overview of total accidents and casualties
* Severity-based analysis
* Vehicle type analysis
* Time-based trends (monthly/yearly)
* Interactive filtering options

## Key Insights

* Most accidents occur during peak traffic hours
* Certain vehicle types are more frequently involved
* Accident severity varies based on location and conditions
* Trends show variation over time

## Outcome

An interactive and user-friendly dashboard was successfully created, providing clear insights into road accident data.
