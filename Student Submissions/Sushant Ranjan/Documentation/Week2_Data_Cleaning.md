# Week 2 - Data Cleaning

## Objective

The objective of this week was to clean the raw dataset and prepare it for analysis.

## Activities Performed

* Removed null and missing values from important columns
* Removed duplicate records using unique identifiers
* Renamed columns for better readability and consistency
* Fixed incorrect data types (date, numeric, categorical)
* Removed unnecessary or irrelevant columns
* Replaced coded values with meaningful labels where required

## Tools Used

* Power BI (Power Query Editor)

## Challenges Faced

* Handling missing values in multiple columns
* Understanding coded data (e.g., numeric values representing categories)
* Ensuring consistency across different tables

## Outcome

The dataset was successfully cleaned and structured, making it ready for further transformation and analysis.
