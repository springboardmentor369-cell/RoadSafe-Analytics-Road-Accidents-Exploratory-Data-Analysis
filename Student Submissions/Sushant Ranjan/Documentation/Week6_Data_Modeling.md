# Week 6 - Data Modeling

## Objective

The objective of this week was to establish relationships between different datasets and build a structured data model for analysis.

## Activities Performed

* Connected Accidents, Casualties, and Vehicles tables
* Used Accident_Index as the primary key to link tables
* Verified relationships for correctness and consistency
* Organized the data model to avoid ambiguity

## Tools Used

* Power BI (Model View)

## Challenges Faced

* Identifying the correct relationships between tables
* Ensuring there are no duplicate or incorrect joins
* Managing multiple tables efficiently

## Outcome

A well-structured data model was created, enabling efficient data analysis and supporting dashboard development.
