In the second week, we learned the fundamentals of Power BI, including its three main views: Report View (used to create dashboards and visualizations), Data/Table View (used to see data in rows and columns), and Model View (used to manage relationships between tables). The instructor introduced importing data from Excel. He clarified the difference between Load (importing data directly) and Transform Data (cleaning and modifying data before loading). In Power Query Editor, we practiced data cleaning steps:
removing duplicates by right-clicking a column such as Customer ID → Remove Duplicates; 
trimming text using Transform → Format → Trim; 
replacing values using Replace Values with Advanced Options to modify specific characters only;
handling missing values by filtering or replacing nulls;
deleting unnecessary rows or columns by right-click → Remove;
splitting columns using Split Column by Delimiter (comma, dot, etc.);
renaming columns by double-clicking the header; 
observing all applied steps in Query Settings. We also learned that drilling means creating a new detailed report page from a visual for deeper analysis.
